# Dashboard
Used Firebase for authentication (registration, authorization),
Axios,
Vue2,
VueRouter,
Vuex,
Vue-notification.

The rest of the components are custom.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
